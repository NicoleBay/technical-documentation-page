Introduction

This documentation provides a comprehensive guide on various web development technologies, including HTML, CSS, JavaScript, Bootstrap, and Figma. Each section covers the essential properties, features, and usage examples to help you understand and effectively use these technologies.

HTML Properties

HTML (HyperText Markup Language) is the standard language for creating web pages. Here are some basic elements:

  • <h1> to <h6>: Heading tags
  • <p>: Paragraph tag
  • <a>: Anchor tag for links
  • <img>: Image tag

HTML elements can have attributes to provide additional information:

  • href: Specifies the URL for an anchor tag
  • src: Specifies the source file for an image
  • alt: Provides alternative text for an image
  • class: Assigns one or more class names for CSS styling 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sample HTML</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a sample HTML document.</p>
    <a href="https://example.com">Visit Example.com</a>
</body>
</html>
CSS Properties

CSS (Cascading Style Sheets) is used to style HTML elements. Here are some basic properties:

  • color: Sets the text color
  • background-color: Sets the background color
  • font-size: Sets the size of the font
  • margin: Sets the outer space around an element
  • padding: Sets the inner space within an element

CSS selectors are used to select the HTML elements to be styled

  • .class: Selects all elements with the specified class
  • #id: Selects the element with the specified ID
  • element: Selects all elements of the specified type 
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
}

h1 {
    color: #333;
    text-align: center;
}

p {
    color: #666;
    margin: 1em 0;
}
JavaScript Properties

JavaScript is a programming language used to create dynamic and interactive effects within web browsers. Here are some basic syntax elements:

  • var, let, const: Declaring variables
  • function: Defining a function
  • if, else: Conditional statements
  • for, while: Looping statements

JavaScript can be used to manipulate the DOM (Document Object Model):

  • document.getElementById(): Selects an element by its ID
  • document.querySelector(): Selects the first element that matches a CSS selector
  • element.innerHTML: Sets or gets the HTML content of an element
  • element.addEventListener(): Attaches an event handler to an element 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sample JavaScript</title>
</head>
<body>
    <h1 id="title">Hello, World!</h1>
    <button id="changeTitle">Change Title</button>

    <script>
        document.getElementById('changeTitle').addEventListener('click', function() {
            document.getElementById('title').innerHTML = 'Title Changed!';
        });
    </script>
</body>
</html>
Bootstrap Properties

Grid System

Bootstrap is a popular CSS framework for developing responsive and mobile-first websites. The grid system is one of its key features:

  • .container: Fixed-width container
  • .row: Creates a row of columns
  • .col: Defines a column within a row

Components

Bootstrap provides a variety of components:

  • .btn: Button styles
  • .navbar: Navigation bar
  • .card: Card layout
  • .modal: Modal dialogs
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Sample Bootstrap</title>
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
</head>
<body>
    <div class="container">
        <div class="row">
            <div class="col">
                <h1>Hello, World!</h1>
                <button class="btn btn-primary">Primary Button</button>
            </div>
        </div>
    </div>
</body>
</html>
Figma Properties

Figma is a web-based UI/UX design tool used for creating interface designs, prototypes, and collaborative work. Here are some essential properties and features:

Figma offers a range of features to streamline the design process:

  • Frames: Define the boundaries of your design (similar to artboards in other tools).
  • Components: Reusable design elements that can be used across multiple frames and projects.
  • Prototyping: Create interactive prototypes to simulate user interactions.
  • Collaboration: Real-time collaboration with team members, allowing multiple users to work on the same project simultaneously.
  • Version History: Track changes and revert to previous versions of your design.

Design Elements

Key design elements in Figma include:

  • Shapes: Basic geometric shapes like rectangles, circles, and lines.
  • Text: Add and style text elements with various fonts and formatting options.
  • Images: Import and manipulate images within your design.
  • Grids and Layouts: Utilize grids and layout tools to maintain consistent spacing and alignment.

Figma example

Here's a basic example of creating a button component in Figma:

  1. Open Figma and create a new frame.
  2. Draw a rectangle for the button background.
  3. Add a text element on top of the rectangle for the button label (e.g., "Click Me").
  4. Select both the rectangle and text, then group them together.
  5. Convert the group into a component by right-clicking and selecting "Create Component".

Now, you can use this button component across your designs and update all instances simultaneously by editing the main component.

Collaborative Features

Figma's collaborative features include:

  • Comments: Add comments directly on the design for feedback and discussion.
  • Sharing: Share your design with team members via a link with view or edit permissions.
  • Team Libraries: Create shared libraries of components and styles for consistent use across projects.
